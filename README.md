# agente-langchain-pytest
Agente Gerador de Testes Unitários

Este projeto utiliza IA Generativa para automatizar a criação de testes unitários em Python, garantindo maior cobertura e qualidade de código.

Tecnologias:

Python 3.10+

LangChain: Orquestração do fluxo da IA.

Azure OpenAI: Modelo de linguagem (GPT).

Pytest: Framework de testes.

Como rodar:

Clone o repositório.

Instale as dependências: pip install -r requirements.txt.

Configure o arquivo .env com suas credenciais do Azure.

Execute o agente: python main.py.

Rode os testes gerados: pytest.
